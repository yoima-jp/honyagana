# Honyagana

`Honyagana` is a resource pack for Minecraft Java Edition that transforms item visuals and the title logo into a soft, distinctive hiragana-inspired style.

Instead of only changing names or adding small UI tweaks, this pack gives your inventory a completely different feel at a glance. Items become more playful, more recognizable, and far more memorable in screenshots, videos, and everyday play.

## 概要

`Honyagana` は、Minecraft Java Edition のアイテム表示やタイトルロゴを、ほにゃがなテイストに変えるリソースパックです。

名前だけを変えるのではなく、インベントリを開いた瞬間に空気が変わるような見た目を目指しています。スクリーンショット、動画、配信でも印象に残りやすく、普段のプレイでも新鮮さを感じられるのが特徴です。

## このプロジェクトが追加するもの

- 多数のアイテムテクスチャを、ほにゃがな風の文字ベースデザインに変更
- ブロックを手に持った時やインベントリに入れた時だけ、専用のほにゃがな表示に変更
- タイトル画面のロゴを専用デザインに変更
- フォントをリソースパック内に同梱し、単体で導入できる構成を用意

## 主な機能

- `アイテム表示の変化`
  通常アイテムだけでなく、ブロック由来のアイテム表示もほにゃがな向けの見た目に差し替えています。

- `設置時の見た目はなるべく維持`
  ワールド内の視認性を保ちつつ、手持ちやインベントリで個性が出るようにしています。

- `タイトル画面まで雰囲気を統一`
  ゲーム開始前から、このパックならではの見た目を感じられます。

- `導入しやすい`
  必要なフォントを同梱しているため、そのまま使えます。

## このパックを入れたくなる理由

- インベントリを見るのが楽しくなる
- アイテムの印象が大きく変わって新鮮に感じられる
- スクリーンショットや配信で個性が出る
- 雰囲気はしっかり変わるのに、普段使いしやすい

## What this project adds

- Reworked item textures in a Honyagana-style text-based design
- Special item-only visuals for many block items while keeping placed blocks recognizable
- A custom title logo that changes the Minecraft title screen atmosphere immediately
- A self-contained pack structure with the font included inside the resource pack

## Features

- `Transforms item visuals`
  Many regular items and block items are redesigned so your inventory feels fresh and unique.

- `Keeps placed blocks familiar`
  The pack focuses on item and inventory presentation, so the world itself remains easier to read while still gaining personality in menus and hand-held views.

- `Changes the title screen too`
  The custom logo helps the pack feel complete from the moment the game starts.

- `Easy to install`
  The required font is bundled with the pack, so the resource pack is ready to use as-is.

## Why download it?

- It makes opening your inventory genuinely fun
- It gives Minecraft a strong visual identity without replacing everything
- It stands out well in screenshots, streams, and videos
- It adds charm and humor while staying practical enough for normal play
- It feels different immediately, even if you already use other visual packs

## Compatibility

- Minecraft Java Edition `1.21.11`
- Resource pack only

## Installation

1. Download the released resource pack.
2. Put it into your `resourcepacks` folder.
3. Enable `Honyagana` from Minecraft's resource pack menu.
